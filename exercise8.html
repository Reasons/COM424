Exercise 8

Color

For Exercise 8, revisit the work you completed for Exercise 5: Symmetry and Asymmetry. Select the composition you created for Exercise 5 that you think was most successful and revise using the design strategy of color to add the dimension of depth to your design. If you already introduced color in your work for Exercise 5, try adjusting the color scheme to produce a different visual impression from the design. Rather than using the entire design you created for Exercise 5, crop the design to an visually interesting detail where adjusting color creates the most visual impact for the viewer.
